HTML Documentation
Tag Usage Attribute
<div> Defines a division or a section class, id, style
<p> Defines a paragraph class, id, style
<a> Defines a hyperlink href, target, title
<img> Embeds an image src, alt, width, height
<h1> to <h6> Heading tags (h1 is the largest, h6 is the smallest) class, id, style
<ul> Defines an unordered list class, id, style
<li> Defines a list item within an ordered or unordered list class, id, style
<table> Defines a table class, id, style, border
<tr> Defines a table row class, id, style
<td> Defines a table cell (data) class, id, style, colspan, rowspan
<header> Defines a header section class, id, style
<nav> Defines a navigation menu class, id, style
<article> Defines an article or a piece of content class, id, style
<section> Defines a section in a document class, id, style
<aside> Defines content aside from the content it is placed in class, id, style
<footer> Defines a footer section class, id, style
<div> Defines a generic container class, id, style
CSS Selector Documentation
Selector Usage Example
element Selects all instances of the specified HTML element p { color: blue; }
#id Selects a specific element with the given ID #header { font-size: 18px; }
.class Selects all elements with the given class .highlight { background-color: yellow; }
element, element Selects multiple elements h1, h2 { font-family: 'Arial', sans-serif; }
element element Selects all elements that are descendants of a specified element ul li { list-style-type: square; }
element > element Selects all elements that are a direct child of a specified element nav > a { text-decoration: none; }
element + element Selects the element that is immediately preceded by the specified element p + p { margin-top: 10px; }
element ~ element Selects all elements that are siblings of a specified element h2 ~ p { font-style: italic; }
:hover Selects and styles an element when it is being hovered over a:hover { color: red; }


CSS Property Documentation
Property Usage Example
color Sets the color of text h1 { color: #333; }
font-size Sets the size of the font p { font-size: 16px; }
font-family Sets the font family for text body { font-family: 'Arial', sans-serif; }
font-weight Sets the thickness of characters in a text strong { font-weight: bold; }
text-align Aligns text within an element div { text-align: center; }
background-color Sets the background color of an element section { background-color: #f0f0f0; }
margin Sets the margin space around an element img { margin: 10px; }
padding Sets the padding space inside an element div { padding: 20px; }
border Sets the border of an element hr { border: 1px solid #ddd; }
display Sets the display behavior of an element ul { display: flex; }
